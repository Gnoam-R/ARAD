## 🔍 map(Google)
우선 총 3가지 지도 API(google, kakao, naver)중에서 구글 맵을 선택한 이유를 말씀 드리자면 
- 다양한 API & 개발 자료 보유
- 전 세계적인 데이터베이스 보유
    - ARCore Geospatial API 기반 좌표 데이터를 사용해 여러 나라에서 사용 가능
- 다양한 map 커스텀 기능
